# Writing-Room Schema

## Domain
Multi-project fiction vault — novels, worldbuilding, and serial fiction across genres (dark romantasy, sci-fi, urban noir, fantasy, slice-of-life, harem fantasy).

## Conventions
- File names: Title Case with spaces allowed (matching existing convention — do not rename existing files to kebab-case)
- Each project gets its own top-level folder with a `README.md` entry point
- Project structure: `Characters/`, `Lore/`, `Scenes/` or `Chapters/`, `Plots/`
- Every project must have: `README.md` (vault guide), `Story Hook.md` (premise), `Story Pitch.md` (elevator pitch)
- Use `[[wikilinks]]` for cross-references within project docs where possible
- `_archive/` folders hold deprecated drafts — never delete, always archive
- `_editor-notes/` holds external editorial feedback, preserved for reference

## Project Status Taxonomy
- **active**: Currently being written or planned. Has recent git activity.
- **dormant**: Concept exists but work has stalled. May resume.
- **seed**: Early-stage. Has a hook/pitch but minimal content.
- **archived**: Superseded or abandoned. Content moved to `_archive/`.

## Genre Tags
- `dark-romantasy`, `sci-fi`, `urban-noir`, `fantasy-worldbuilding`, `slice-of-life`, `harem-fantasy`, `isekai`, `litrpg`

## Content Types
- **Characters**: Deep profiles with motivations, contradictions, subtextual arsenal
- **Lore**: Worldbuilding — magic systems, rules, history. Must pass the "So What?" test (how does this make the protagonist's life harder?)
- **Scenes/Chapters**: Drafts and structural audits
- **Plots**: Story arcs. Every arc must answer: *What does the protagonist lose?*

## Page Thresholds
- **Create a project folder** when there's a Story Hook + at least one character or scene
- **Create a lore file** when a worldbuilding element affects the plot (not just flavor)
- **Archive a project** when it's been untouched for 90+ days and has no active scenes
- **Split a file** when it exceeds 300 lines

## Lyra's Rules (from _editor-notes/)
Carried forward from the previous content editor:
1. No "nice" notes — if a file hasn't been updated in 30 days, archive or delete
2. Templates are law — customize but never skip fields
3. The "So What?" Test — every lore note must affect the protagonist
4. Backlinks > Folders — if you can't link it, it doesn't exist
